Now if you are using a Linux system such as Debian, Ubuntu or Red Hat, the central processing unit is controlled by the Linux Kernel. Learn about Linux from the [Linux Foundation](https://trainingportal.linuxfoundation.org/courses/introduction-to-linux-lfs101).

> At the heart of every Linux system is the [Linux kernel](https://git.kernel.org/), which is the core part of the operating system. The kernel manages communication between software and hardware. It controls resources such as the CPU, memory, and connected devices, and it coordinates the programs running on your system. It looks like  [this](https://git.kernel.org/pub/scm/linux/kernel/git/torvalds/linux.git/tree/?id=v7.2-rc4).
